# Algo 17: Pivot Point Breakout Strategy

## Overview
This is a live trading algorithm that uses pivot points calculated from recent price data to enter long/short positions on breakouts above resistance levels (R1/R2) or below support levels (S1/S2). It includes take-profit (+0.5%) and stop-loss (-1%) exits, with pivot recalculation after each trade.

The algorithm reads live price data from a JSON file (`price_data.json`) generated by a simulated exchange feed.

## Prerequisites
- Python 3.x installed
- Both scripts (`Exchange.py` and `Algo 17.py`) must be run from the same directory: `Algo 17 Pivot Point Breakout`

## Running the Scripts

### 1. Start the Price Feed (`Exchange.py`)
This script simulates a live price feed by continuously updating `price_data.json` with random walk prices.

**Command:**
```bash
python Exchange.py --start 1000 --interval 1.0 --volatility 0.002 --drift 0.0
```

**Parameters:**
- `--start` (float, default: 1000.0): The initial price written to `price_data.json`.
- `--interval` (float, default: 1.0): Seconds between price updates (e.g., 0.5 for faster updates).
- `--volatility` (float, default: 0.002): Price volatility as a fraction of the current price (e.g., 0.002 = 0.2% per tick).
- `--drift` (float, default: 0.0): Optional directional bias per tick (positive for upward drift).

**Flow:**
1. Initializes `price_data.json` with the starting price.
2. Enters an infinite loop:
   - Generates a new price using random normal distribution based on volatility and drift.
   - Writes the updated price to `price_data.json`.
   - Sleeps for the specified interval.
3. Continues until manually stopped (Ctrl+C).

### 2. Run the Trading Algorithm (`Algo 17.py`)
This script reads prices from `price_data.json`, calculates pivot points, and executes trades based on breakout logic.

**Command:**
```bash
python "Algo 17.py"
```

**No parameters** – all settings are hardcoded in the script (warm-up count, TP/SL, etc.).

**Flow:**
1. **Warm-up Phase:** Collects 20 prices from `price_data.json` (sleeps 2 seconds between reads).
2. **Pivot Calculation:** Computes pivot levels (P, R1, S1, R2, S2) from the warm-up prices.
3. **Trading Loop:**
   - Reads current price every 2 seconds.
   - Displays pivot levels.
   - Checks for entry: Long if price breaks above R1/R2, Short if below S1/S2.
   - If in position, monitors for TP (+0.5%) or SL (-1%) exits.
   - After exit, recalculates pivots from the last 5 prices.
4. Continues indefinitely until stopped (Ctrl+C), printing final PnL.

## Stopping the Scripts
- Press `Ctrl+C` in each terminal to stop the respective script.
- The price feed (`Exchange.py`) must be running for `Algo 17.py` to receive live prices.

## Notes
- Run `Exchange.py` first in one terminal, then `Algo 17.py` in another.
- Prices are simulated; in a real setup, replace `Exchange.py` with actual market data feed.
- Adjust `--interval` and `--volatility` for faster/slower/more volatile price updates.
- The algorithm uses fixed units (100) and assumes 1:1 position sizing.
